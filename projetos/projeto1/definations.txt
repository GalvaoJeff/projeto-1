Sistema de Gerenciamento via Terminal

Crie um sistema de gerenciamento que rode no terminal usando PHP puro (sem frameworks). ok
O sistema deve ter um menu interativo com as seguintes funcionalidades:

- **Cadastrar** um novo registro com pelo menos 4 campos (sendo pelo menos 1 string, 1 inteiro e
1 booleano). ok
- **Listar** todos os registros cadastrados, ordenados alfabeticamente. ok
- **Buscar** registros por nome (busca parcial e case-insensitive). ok
- **Editar** um registro existente (permitindo manter valores atuais ao pressionar Enter). ok
- **Remover** um registro com confirmação antes de excluir. ok
- **Estatísticas** — exibir pelo menos 3 informações calculadas a partir dos dados (ex: total, média,
contagem por status). ok

### Regras

1. Os dados devem ser armazenados em um **array de arrays associativos** - ok
2. Cada registro deve ter um **ID único** gerado automaticamente - ok
3. Todos os inputs do usuário devem ser **validados** antes de salvar - ok
4. O código deve ser separado em dois arquivos ou mais - ok
5. O menu deve rodar em **loop infinito** até o usuário escolher sair - ok
